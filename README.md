# AI Mood Journal (Web App)

A simple web app for students to track their daily mood with basic AI sentiment detection.

---

## Why Students Love It
- Mental-health themed, encouraging self-reflection  
- Simple AI sentiment analysis (keyword-based or mock AI)  
- Clean, pastel-themed user interface  

---

## Features
- Users can type their daily mood or feelings  
- App provides a simple **Mood Summary**: `Happy / Stressed / Neutral`  
- Entries are saved in **localStorage** for easy tracking  
- Cute pastel UI for a pleasant experience  

---

## Tech Stack
- **HTML** – App structure  
- **CSS** – Styling, pastel colors, and layout  
- **JavaScript** – Interactivity, sentiment detection, localStorage  
- **No backend required** – Runs fully in the browser  

---

## How to Use
1. Open `index.html` in your browser  
2. Type your mood or feelings in the input field  
3. Click **Submit** to see your mood summary  
4. Previous entries are saved locally  

---

## Notes for Collaborators
- Frontend-only project; easy to extend  
- Sentiment detection can be upgraded with a proper AI model in the future  
- Styles are customizable via `style.css`
